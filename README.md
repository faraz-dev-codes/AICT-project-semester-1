# Delicious Bites Restaurant

## Project Overview
This project is a **frontend restaurant website** developed as part of my **AICT 1st Semester** coursework.  
It is built using **HTML, CSS, and JavaScript** and includes sections such as menu display, ordering interface, and basic user interaction.  
The project focuses on **responsive design**, **clean UI**, and **proper file structure**.

---

## Features
- **Multiple Pages:**  
  - Main/Homepage  
  - Login/Signup page  
  - Menu page  
  - Order page
- **Responsive Design:** Works on desktop, tablet, and mobile devices.
- **User Interaction:** Users can browse the menu and place orders.
- **Clean UI/UX:** Modern and user-friendly interface.

---

## Technologies Used
- **HTML5** – Structure of the web pages
- **CSS3** – Styling and layout
- **JavaScript** – Basic interactivity and dynamic features
